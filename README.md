# css-inliner

## Description

## Usage
* $ css-inliner /path/to/file-to-inline.html

### Sublime Text Build:  
* Place CSS Inliner.sublime-build under Packages/User/  
* Select Tools > Build System > CSS Inliner  
* CTRL + b to inline current file  

## Install
* $ pip install -r requirements.txt
